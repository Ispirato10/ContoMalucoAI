# **App Name**: AdVision AI

## Core Features:

- Product Content Upload & Management: Allow users to upload product images, input product name, description/benefits, and optional product links, storing these details in Firestore.
- Creative Campaign Selection: Enable users to choose from predefined ad campaign themes (e.g., 'Luxury', 'Seasonal', 'Industry-specific') and target social media formats (e.g., 'Instagram Story', 'Feed Post', 'Banner').
- Intelligent Ad Prompt Generation Tool: An AI-powered tool that synthesizes user inputs (product details, selected theme, format) and predefined advertising styles (e.g., 'luxurious style', 'professional lighting') into highly specific, persuasive prompts for the image generation AI.
- Generative AI Ad Image Creation: Invoke an external image generation AI service (like OpenAI's DALL-E) to produce professional, high-conversion advertising visuals based on the constructed prompt.
- Interactive Image Preview & Download: Display generated advertising images in a high-fidelity preview, allowing users to inspect the results and download chosen assets.
- Thematic Template System: Manage a backend collection in Firestore containing configurable templates for various themes and styles, guiding the AI's creative output for different platforms and aesthetics.

## Style Guidelines:

- Primary color: A deep, professional blue-purple (#2E2E6B), reflecting sophistication and technology in a dark theme.
- Background color: A subtle, very dark grey with a hint of the primary hue (#222226), providing a premium backdrop.
- Accent color: A vibrant, elegant magenta-purple (#A857DB), used for highlights and calls to action, creating impactful contrast.
- Headline font: 'Belleza' (humanist sans-serif) for an artistic, design-forward and elegant appeal. Body text font: 'Alegreya' (humanist serif) to ensure readability and complement the elegant headline.
- Use sleek, minimalist, and modern icons that align with a premium advertising and AI-driven creative tool, avoiding overly ornate designs.
- Implement a clean, organized, and intuitive layout that prioritizes content upload and generated image display, minimizing visual clutter.
- Incorporate subtle and smooth transitions for key user interactions, such as image generation completion, selection changes, and navigation, to enhance the sophisticated user experience.
